📱 راهنمای نصب در ترمکس:

1. نصب پیش‌نیازها:

pkg update && pkg upgrade pkg install python python-pip git

2. ایجاد فایل ربات:

mkdir MyBot && cd MyBot nano bot.py

3. کد بالا را در فایل کپی کرده و ذخیره کنید (Ctrl+X, Y, Enter)

4. نصب کتابخانه‌ها و اجرا:

pip install rubpy python bot.py
